# DatePicker 日期选择器

## 用途
用于选择日期，如筛选预约日期。

## 使用方式
```html
<div class="date-picker">
  <input type="text" class="date-picker__input" value="2025/11/29" readonly>
  <span class="date-picker__icon">📅</span>
</div>
```

## 说明
- 点击触发日期选择弹窗
- 显示格式：YYYY/MM/DD
- 带日历图标

## 预约管理页使用
筛选区域包含日期选择器，用于选择查看特定日期的数据。

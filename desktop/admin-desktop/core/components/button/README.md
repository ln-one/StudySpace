# Button 按钮组件

## 用途
通用按钮组件，支持多种变体和尺寸。

## 使用方式
```html
<!-- 主按钮 -->
<button class="btn btn--primary">保存规则</button>

<!-- 次要按钮 -->
<button class="btn btn--secondary">取消</button>

<!-- 文字按钮 -->
<button class="btn btn--text">详情</button>

<!-- 危险按钮 -->
<button class="btn btn--danger">删除</button>

<!-- 小尺寸 -->
<button class="btn btn--primary btn--sm">小按钮</button>

<!-- 大尺寸 -->
<button class="btn btn--primary btn--lg">大按钮</button>

<!-- 禁用状态 -->
<button class="btn btn--primary" disabled>禁用</button>
```

## 变体
| 类名 | 说明 |
|-----|------|
| `btn--primary` | 主按钮，用于主要操作 |
| `btn--secondary` | 次要按钮 |
| `btn--text` | 文字按钮，用于次要操作 |
| `btn--danger` | 危险按钮，用于删除等操作 |
| `btn--ghost` | 幽灵按钮，透明背景 |

## 尺寸
| 类名 | 说明 |
|-----|------|
| `btn--sm` | 小尺寸 |
| 默认 | 中等尺寸 |
| `btn--lg` | 大尺寸 |

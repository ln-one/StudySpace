## 功能特性

### 用户管理功能
- **添加用户**: 完整的用户信息表单，包括基本信息、专业年级、权限等级
- **编辑用户**: 预填充现有数据，支持修改除学号外的所有信息
- **权限管理**: 快速切换普通用户和VIP用户权限
- **数据导出**: 支持Excel、CSV、PDF格式，可自定义导出内容和筛选条件

### 公告管理功能
- **富文本编辑**: 支持粗体、斜体、下划线、列表、链接等格式
- **发布设置**: 优先级、发布范围、置顶、推送通知、定时发布
- **实时预览**: 编辑时实时预览公告效果
- **草稿保存**: 支持保存草稿，避免内容丢失

### 系统管理功能
- **数据导出**: 导出完整系统数据
- **缓存管理**: 清除系统缓存优化性能
- **系统重置**: 安全的系统重置功能，双重确认保护

## 使用说明

1. **添加用户**: 点击"+ 添加用户"按钮，填写表单信息，选择权限等级
2. **导出数据**: 点击"导出数据"按钮，选择格式和内容，设置筛选条件
3. **发布公告**: 点击"+ 发布公告"按钮，编辑内容，设置发布选项
4. **系统管理**: 在设置页面使用快速操作功能

所有功能都经过测试，确保交互流畅，用户体验良好。

# 座位管理模块

## 概述
座位管理模块提供管理员对学习空间座位的可视化管理功能。

## 页面

### seat-list.html - 座位列表与管理
主要功能：
- **筛选区域**：楼层、区域、日期、时间段筛选
- **座位网格视图**：6x5 网格布局，直观展示座位状态
- **座位详情面板**：编辑座位信息、设施、预约状态

## 座位状态
| 状态 | 说明 | 颜色 |
|------|------|------|
| available | 可用 | 白色 |
| reserved | 已预约 | 橙黄色 |
| occupied | 使用中 | 绿色 |
| disabled | 不可用 | 灰色 |
| selected | 已选中 | 蓝色 |

## 交互说明
1. 点击座位可选中并在右侧面板查看/编辑详情
2. 下拉筛选支持楼层、区域、日期、时间段
3. 保存按钮会更新座位状态
4. 移动端详情面板从底部滑出

## 设计规范
- 遵循 Apple Glassmorphism 设计风格
- 使用毛玻璃效果、圆润边角、精致阴影
- 支持响应式布局（桌面/平板/手机）
- 座位入场动画增强用户体验

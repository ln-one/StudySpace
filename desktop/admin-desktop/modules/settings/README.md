# 管理员设置模块

## 概述
管理员设置模块提供系统配置和管理功能，涵盖图书馆座位预约系统和 GTD 任务管理两大核心功能。

## 页面

### index.html - 设置主页面

## 功能模块

### 1. 管理员信息
- 头像、姓名、邮箱、手机号
- 密码修改
- 点击卡片弹出编辑弹窗

### 2. 预约设置
- 单次预约最长时间（1-8小时）
- 每日最多预约次数
- 允许提前取消的时间
- 最早可提前预约天数
- 签到宽限时间
- 临时离开最长时间
- 签到方式（扫码/位置/手动）
- 自动释放座位开关

### 3. 权限设置
- 用户角色权限矩阵
- 普通用户 / VIP用户 / 管理员
- 预约、管理、查看统计等权限控制

### 4. 违约处理
- 违约类型：未签到、迟到超时、提前离开未签退
- 处罚方式：警告、禁止预约（1/3/7天）
- 月度违约上限
- 超限处罚
- 违约记录保留时间

### 5. 通知设置
- 用户通知开关（预约成功、签到提醒、即将结束、违约通知、任务到期）
- 通知渠道（站内消息、邮件、短信、微信推送）

### 6. 图书馆设置
- 基本信息（名称、电话、地址）
- 开放时间（工作日/周六/周日）
- 楼层与区域管理

### 7. GTD 设置
- 任务设置（默认优先级、提醒时间、过期处理、每日上限）
- 学习记录（自动计时、目标周期、默认目标、排行榜）
- 番茄钟设置（专注时长、短休息、长休息、间隔）

```
settings/

├── styles/

└── README.md
```

## 交互说明
1. 左侧菜单切换不同设置区块
2. 管理员卡片点击弹出编辑弹窗
3. 每个区块独立保存/重置
4. 开关类设置即时生效
5. 响应式布局适配移动端
